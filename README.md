## APIs

### Wikidata SPARQL Query Service API
The Wikidata SPARQL Query Service provides a public SPARQL 1.1 endpoint for querying the Wikidata knowledge graph. Supports complex entity lookups, relationship traversal, federated queries, and structured data export covering millions of entities across people, organizations, places, and concepts.

**Human URL:** [https://www.wikidata.org/wiki/Wikidata:SPARQL_query_service](https://www.wikidata.org/wiki/Wikidata:SPARQL_query_service)


#### Tags:

 - Knowledge Graph, SPARQL, Linked Data, Open Data

#### Properties

- [Documentation](https://www.wikidata.org/wiki/Wikidata:SPARQL_query_service)
- [DeveloperTools](https://query.wikidata.org/)

### Wikidata MediaWiki API
The Wikidata MediaWiki API provides programmatic access to Wikidata content including entity retrieval, editing, searching, and administrative operations. Supports reading and writing Wikidata items and properties using the wikibase data model.

**Human URL:** [https://www.wikidata.org/w/api.php](https://www.wikidata.org/w/api.php)


#### Tags:

#### Properties

### Wikidata Linked Data API
The Wikidata Linked Data API provides entity data in multiple RDF serialization formats including JSON-LD, Turtle, N-Triples, and RDF/XML via the Special:EntityData endpoint. Enables content negotiation for Linked Data access to any Wikidata item or property.

**Human URL:** [https://www.wikidata.org/wiki/Wikidata:Data_access](https://www.wikidata.org/wiki/Wikidata:Data_access)


#### Tags:

 - Knowledge Graph, Linked Data, RDF, JSON-LD, Open Data

#### Properties

- [Documentation](https://www.wikidata.org/wiki/Wikidata:Data_access)
- [OpenAPI](openapi/wikidata-mediawiki-openapi.yml)

### Wikidata / DBpedia SPARQL API
Wikidata and DBpedia provide SPARQL endpoints for structured knowledge graph queries. APIs enable entity lookup, relationship traversal, property retrieval, and structured data export covering people, organizations, places, and concepts across the Wikimedia knowledge base.

**Human URL:** [https://www.wikidata.org/wiki/Wikidata:SPARQL_query_service](https://www.wikidata.org/wiki/Wikidata:SPARQL_query_service)
