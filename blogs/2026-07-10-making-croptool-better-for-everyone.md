CropTool is a browser-based image editing tool used daily by Wikimedia Commons contributors to crop and improve media files. Recently Vera de Kok carried out a major redesign of the tool, adding mobile support, multilingual interfaces, improved metadata handling, and a more modern user experience. The result is a faster and more intuitive workflow that helps contributors improve media while keeping Wikimedia Commons' structured data accurate.
